    Payment Gateway Changelog

    1.2.2

    Refactoring

    Refactoring of functionality to provide better support for submitting non-standard record types for payment.

    Account No. on Payment Gateway Log

    Include the corresponding Account No. in the Payment Gateway Log table.

    1.2

    Realex Support

    Support for new Payment Provider, Realex.

    1.1.6

    Taking Payments on Cash Receipt Journal

    Support for taking a payment through Payment Gateway direct from the Cash Receipt Journal page.

    Address Details

    Bug fix: billing and delivery addresses switched when presented to payment provider.

    1.1.5

    Suggest Payment Gateway Payment Report

    New action added to the Cash Receipt Journal page to create journal lines from Payment Gateway Log records. Also attempt to apply the journal line to the correct posted invoices.

    Missing Permission on Service Password Table

    Explicit permission needs to be granted to the Service Password table from the Payment Gateway Setup page. Added in this version.

    1.1.4

    Default Configuration Support

    Default Configuation action added to Payment Gateway Setup page to retrieve default service URL and other data from configuration service.
