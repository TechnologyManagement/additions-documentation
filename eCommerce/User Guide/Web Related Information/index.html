    Web Related Information

    Back on the Web Site Items Page Web Related Information allows you to add additional information to a web item. This can be a web link, free text or an attachment file. The additional information displays on the web site under a collapsible tab under the main details of the item.

    To access the Web Related Information page, navigate to the Web Site Item list.

    Select the relevant item and click the Web Related Information option from the ribbon. The Web Site Item list also shows the number of related information records in the list, clicking the field also open the Web Item Info Page.

    The information placed in this table against the website item will show under a heading Show Additional Information against the item on the website.

    The first field to fill in is the Information Type Code, this determines the type of additional information you want to show:

    • File –a file attachment in any type of file-format: word document to PDF to even a simple text file.

      • Once this code is selected, use the ‘Upload file’ in the ribbon to import the file.

    • Link – a web link which their customers can click on; it may be a completely different website or to another page on the current site.

    • Text – to type free text to appear in the additional information add in the text to the details box.

    • HTML – HTML code can be entered. This can be used to add extra content to the page. E.g an embedded YouTube video demonstrating the product.

    This check box Authenticated Viewable Only limits the visibility of additional info to registered web site users only

    Like the web site categories there is a Sort Order field, this is the order the additional information appears on the website. The Move Up & Down arrows in the ribbon can be used to change the order.
