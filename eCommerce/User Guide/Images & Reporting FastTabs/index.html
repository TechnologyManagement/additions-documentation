    Images & Reporting FastTab

    The next two FastTabs are very self-explanatory the first is the Images FastTab

    You can choose how images which have been uploaded to the website are stored. The options are Database, Network or Default.

    With Database the images are stored within the Microsoft Dynamics NAV database itself.

    Network allows you to select a network folder to save the images to. If you choose this option you must then fill in the field Image Network Folder, to specify the location you want them held in.

    Default means the image storage method on the eCommerce set up is used.

    The second tab is the Reporting FastTab this is where the criteria is set for creating customer statements when requested through the My Account Portal.

    The Statement Report ID can be changed if you have a bespoke Customer Statement designed, and you can start and end dates for the statement with the Start/End Date Calculation fields.

    By clicking on the Statement Request Page button in the ribbon. It will open the reports request page where further filters such as below can be applied on top of the date filters you applied earlier.
