    Web Site Categories

    The user guide has previously mentioned Web Item Categories and how you can link Items to Web Categories, it is still be addressed as to how those categories appear in the first place, so you can make the link. To get to the Web Site Categories page you can use the search in your role centre to navigate to the page.

    From this screen you have several different options to choose from on the ribbon as well as several different fields on display giving you information on each category.

    The first thing you will notice is that some codes are highlighted in bold font whereas others aren’t. This is to identify the Parent Category (bold) and the Child Category (not bold) within it.

    You can drill into the category codes by simply highlighting the code you want to view and clicking edit. From this screen you can also create new Web Site Categories. To do this simply click on the New button which will open an Item Category Card.

    NB: Item Categories when created will also create a record in the Web Site Categories list. It is in the Web Site Categories list that you will need to edit each category based on the extra fields related to our eCommerce solution.

    On the Item Category Card, you can fill in details such as Code (unique identifier), Description and whether the category you are creating is a child category, if it is fill in the field Parent Category and choose which category your new category belongs to.

    From the Web Site Categories page, you can add items to categories based on filters. This way is more useful when completing the initial set up as many items can be added to categories quickly.

    From the Web Site Category page, select the category you would like to add items to, and select Items from the ribbon and then go to the Actions tab in the ribbon.

    There are three options for adding multiple items to categories at once;

    • Add Items from List; Will open the item list and allow you to select individual items to add or highlight multiple items by holding CTRL or CTRL SHIFT as you click new items.

    • Add Items from Filter; items can be added based on fields on the item card. For example, items within an item number range. The below screenshot shows how to Add all items within the 1000 to 1750 item number range.

    • Add Nonstock Items from Filter; uses the same method as Add Items from Filter but is specific to Nonstock items only.

    Other actions on the ribbon include; Include and Remove Selected, which when either is pressed will flag the category highlighted as either Show on Website or not. This can also be done manually by simply clicking on the field. The reason for having this flag is so that you can control what products you want visible online and which you don’t.

    NB: If you don’t plan on showing a category and you have web items in that category, please make sure you flag the website items within that category to also NOT Show on Website.

    The options to Move Up and Move Down will move the highlighted category up or down in the list. If it is a Parent Category (category in bold text) that you are moving then it will not only move that highlighted line, but all child categories associated with it. If you move a child category up or down, it will only allow you to move within the category you have associated it with.

    When moving a category up and down you will notice that the field Display Order changes. The Display order determines in what order the categories will appear on the website. So rather than in alphabetical order or the order in which you created them in, you may want to do it by most popular category at the beginning followed on by less popular ones.

    If you click on the button labelled Items, you will be presented with a list of Items/ Nonstock Items that are associated with that category or with an empty list where you can then add items to link to that category.

    The same screens will show of you were to drill into the field No. of Items next to each category line.

    The final thing to highlight on the web categories screen is the field Display Area, which is against each category. The display area gives you the ability to change where on the homepage you would like the menu to visible. The field itself is a dropdown field with three options:

    • Menu – Selecting this option will only show the category along the top of your website homepage.

    • Slider – Will instead show the category in the middle of the homepage, you may want to do this to showcase deals within a category or highlight those products more than others.

    • Menu/Slider – Will put the category both at the top of the homepage as well in the centre, giving customers 2 ways in which, they can get to products within a certain category.

    NB: If the category is not flagged to show on website you will not be able to change the display area of it within Microsoft Dynamics NAV.
