    Search Index Rebuild

    Defragmentation of the search index may occur over time when many changes are made to the website, this can adversely affect performance of the website search tool. An Index Rebuild will reorganise the search index with the aim of making it more efficient.

    Symptoms that indicate a search index rebuild may be needed are:

    • partial or no search results are returned.

    • unexpectedly slow searches.

    • items that you expect to appear based on your search criteria aren't displayed.
