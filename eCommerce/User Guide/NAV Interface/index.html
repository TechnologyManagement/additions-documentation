    NAV Interface

    eCommerce setup changes that are made in NAV, such as adding new web site items, adding images, adding categories, are automatically pushed through to the live website every 4 hours.

    If you don’t want to wait you can manually trigger an update using the NAV Interface page of the Admin Area. The Interface update is broken down in to sections, this allows specific areas to be updated separately and makes it easier to determine the cause of update errors.

    Import Settings: will push through changes related to the eCommerce setup pages in NAV. This includes amongst other things, the price break behaviour and rounding precision, stock availability codes and default customer pricing.

    Import Item Menu: will push through changes related to the structure, names and display order of Item Categories.

    Import Images: will push through new images to the web site that have been uploaded in NAV. This includes Item images as well as Item Category images.

    Import Items: will push through new Web Site Items and changes to existing Web Site Items such as descriptions, additional information and prices.

    Send Orders to NAV: if Customers have placed Web Orders that haven’t yet appeared in the Pending Inbound Documents page, this option will attempt to resend to the orders to NAV.

    If any of the NAV interface updates are successful, a green message will appear in confirmation.
